# Comfy Color Scheme

*Comfy* is a warm color scheme for Vim inspired by gruvbox.

**NOTE**: This theme only works with versions of Vim / terminals that have
support for `termguicolors`.

## Installation

Install plugin
```vim
Plug 'jayden-chan/vim-comfy'
```

Set theme
```vim
set termguicolors
colorscheme comfy
```
